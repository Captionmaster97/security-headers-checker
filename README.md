# 🛡️ Security Headers Checker

A lightweight Python tool that analyzes HTTP response headers and identifies missing security headers based on common web security best practices.

---

## Features

- Checks important HTTP security headers
- Simple command-line interface
- Beginner-friendly Python project
- Useful for learning web application security

---

## Security Headers Checked

- Content-Security-Policy (CSP)
- Strict-Transport-Security (HSTS)
- X-Frame-Options
- X-Content-Type-Options
- Referrer-Policy
- Permissions-Policy

---

## Requirements

- Python 3
- requests library

Install dependency:

```bash
pip install requests
```

---

```bash
python security_headers_checker.py
```
